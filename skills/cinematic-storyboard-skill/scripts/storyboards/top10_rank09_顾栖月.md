# 《沸腾之雪》天榜十绝·天榜第九·顾栖月（15s）最终版

## 核心设计理念
- **视觉重心**：素白劲装 vs 纯黑刺客，白瀑青崖间的明暗博弈。
- **武器系统**：羊脂玉笛（攻防核心），墨色青油纸伞（视觉挂件/防御伏笔）。
- **气场设定**：清冷、凌厉、动作干脆（非柔弱感），强调“笛随影动”。

## 场景配置
- **地点**：文成百丈漈（实景参考）。
- **元素**：飞瀑、云气、青藤、苔石、潭水、山风。
