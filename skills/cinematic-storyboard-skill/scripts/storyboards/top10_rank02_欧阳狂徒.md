# 《沸腾之雪》天榜十绝·天榜第三·欧阳狂徒（15s）正式版

## 🎬 核心设计理念
- **视觉逻辑**：**“原始的暴力与毁灭”**。画面充满颗粒感与焦灼感。强调“重刃”的物理重量与劈砍后的视觉爆炸。色彩以焦土地黑、火焰橘、高饱和血红为主。
- **动态节奏**：野兽般的爆发。不讲究章法，每一刀都是倾尽全力的毁灭，强调重击后的震屏感。
- **环境背景**：古战场废墟，残垣断壁，地面满是焦土与尚未熄灭的余火，天空阴云压城。

## 🎥 镜头分解 (15s)
写实高武武侠短片，15 秒，24fps，16:9，35mm 胶片颗粒，电影级光影，无字幕水印 Logo 及现代元素，无法阵符文神光，无跳切瞬移。场景为古战场废墟，断墙残垣，地面焦土带余烬，硝烟弥漫，同一地形连续作战。主角参考 @wild-blade-ouyang，赤裸上身露出虬结肌肉与密布战斗伤疤，下着重型皮革战裙，单手拖行门板大小的黑铁大刀“破军”，具备极致物理重量感。刺客为西燕精锐重甲铁骑或劲装杀手，身形壮硕。音效含沉重金属拖地声、野兽般喘息、震耳欲聋轰鸣声、地面塌陷巨响、骨骼碎裂声、大火爆裂声、号角远鸣。
0-3.0 秒，狂：黑刀破军。镜头为低机位仰拍接近景，宽画幅展现废墟压迫感。主角低头侧行，单手拖着大刀划过焦土，带起成串火星与飞扬尘土，肌肉线条随呼吸起伏，眼神如饿狼。
3.0-6.0 秒，震：大地崩裂。镜头为中景接升格。主角发力跃起，身体在空中拉伸如黑色满弓，双手握刀重重砸入地面。以刀刃为中心，焦土瞬间崩裂，掀起两米高的土浪与环形冲击波，镜头伴随强烈震动。
6.0-9.0 秒，碎：血雾屠戮。镜头为特写转升格。面对刺客围攻，主角横向暴虐轮斩。沉重刀锋瞬间击碎敌人重甲，火星四溅。捕捉血液在极速重击下被震碎成细密红雾的慢动作瞬间，极具视觉冲击力。
9.0-12.0 秒，崩：万夫莫敌。镜头为动态大广角快速穿梭。主角在敌阵中横冲直撞，刀势如黑色旋风，所过之处瓦砾崩飞，盔甲碎片漫天。每一击都具备碎裂反馈，无虚晃招式。
12.0-15.0 秒，定：狂徒之怒。镜头为近景仰拍特写接极速后拉。主角将大刀重重杵入地面，单手抹过满脸血迹，露出一口白牙残忍一笑，背后是硝烟与废墟，定格于主角眼神。
15.0 秒，全景收尾，战场远处的号角声响起，火烧木头的噼啪声清晰可见。
