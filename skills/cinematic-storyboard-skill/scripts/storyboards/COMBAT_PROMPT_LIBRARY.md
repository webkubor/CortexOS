# 沸腾之雪 · 打斗拍摄规则手册

**版本**：v2.1
**用途**：所有打斗场景生成的统一规则，任何人/AI拿此手册即可稳定输出达标打斗戏。
**原则**：打斗戏靠规则体系输出质量，不靠灵感或随性发挥。

---

## 一、硬红线（全武打通用，零容忍）

1. **禁止项**：无字幕/无字卡/无水印/无Logo，无现代元素，无符文神光，无无动机跳切。
2. **人体一致性**：禁止多手、多臂、多武器复制、武器瞬间替换。
3. **武器锁死**：长度/材质/主从关系全程不变，不断裂不变形不脱手不互换。
4. **交手锁死**：指定双人对抗则禁止第三人下场或误入。
5. **禁止文学修辞**：不写"像白鹤掠水""干净到像写字""猫式落地"等比喻。所有动作描述必须是**物理量描述**（身体角度/重心方向/接触部位/位移距离）。
6. **禁止站桩**：全程必须有位移，绝对禁止两人面对面站定后一人出招。

## 二、时间分配黄金法则（15秒版）

### 2.1 时间预算（15秒 = 15000ms，一分不能浪费）

| 时段 | 时长 | 用途 | 禁止 |
|------|------|------|------|
| 起势 | 0-2s | 交代环境+人物就位+首次接触 | 禁止空拍摆pose，禁止超过2s |
| 激战 | 2-13s | **核心打斗时间，全片精华所在** | 禁止大段铺垫，禁止超过1s无攻防交换 |
| 收尾 | 13-15s | 结果定格+余韵 | 禁止拖到超过2s |

### 2.2 激战期硬规则（2-13秒）

- **每2秒至少1次有效攻防交换**（接触/破招/反制 三选一）。
- **单招时长 0.3-0.6秒**，禁止演示感慢动作。
- **连续动作，禁止回合制**：A出一招→B接一招→A再出一招 这种回合制节奏严禁出现。打斗应该是连续的攻防流，双方同时移动同时出手。
- **环境必须参与**：每段至少1个环境微反馈（水花外炸/火焰偏转/雾流切线/石粉震落/竹折雪落/木梁颤动）。

### 2.3 史诗镜头（全片最多1次，打斗中间）

打斗不是只有"啪啪啪打"。在激战高潮处（建议7-10秒区间），可以切一次**壮观大景别镜头**，给观众"哇"的视觉冲击。

**配额**：
- 15秒片：**最多1次**，时长 **1-2秒**
- 位置：必须在打斗中间（7-10秒区间），**禁止用于起势和收尾**

**什么是史诗镜头**：
- 大远景俯拍/高机位全景，展示整个场景的宏大尺度
- 必须包含：环境 + 人物的相对位置关系 + 战斗痕迹
- 示例：剑门关栈道——俯拍整条栈道，火把零星散落、断裂木板悬空、黑衣人倒伏、白色身影在栈道中央移动，深渊雾气从两侧升起

**什么不是史诗镜头**：
- ❌ 纯环境空镜（没人就不叫打斗戏）
- ❌ 起势时的大全景铺垫（浪费黄金时间）
- ❌ 收尾的"拉远俯拍展现孤独背影"（这是文艺片不是打斗片）

**切入/切出规则**：
- 切入：从打斗动作（比如剑锋挥出）直接切到大景别
- 切出：大景别必须以动态收束（比如从全景切回武器接触特写或下一招的起始）
- 禁止在大景别上停留超过2秒，否则打斗节奏断裂

## 三、慢镜头与特写规则

### 3.1 慢镜头（全局配额）

| 时长 | 配额 | 位置 | 用途 |
|------|------|------|------|
| 15s片 | **最多1次**，**0.2-0.35秒** | 必须在**打斗中间** | 捕捉关键打击瞬间 |

**铁律**：
- ❌ 禁止用慢镜头做"起势摆拍"（面部特写+风吹头发+眼神变化 = 浪费）
- ❌ 禁止用慢镜头做"收尾回眸"（剑归鞘+风吹衣袂+眼神平静 = 浪费）
- ✅ 只允许：**武器接触皮肤的瞬间 / 刀锋擦过瞬间的血线 / 剑尖点破衣甲的裂口 / 拳骨撞击关节的震动**
- 慢镜头后必须**极速回正**（0.1s内恢复实时速度），形成冲击感。

### 3.2 特写（分三类，必须标注类型）

| 类型 | 定义 | 时长 | 示例 |
|------|------|------|------|
| **战斗特写** ✅ | 攻击/防御瞬间的微观画面 | 0.2-0.5s | 剑尖划过喉口的血线、刀背格挡时手臂肌肉绷紧、弩箭穿过衣角撕裂布料 |
| **环境特写** | 战斗引发的环境破坏 | 0.2-0.3s | 踩断的竹竿弹起、箭矢钉入木柱箭尾颤抖、刀痕划过石壁火星飞溅 |
| **表情特写** ⚠️ | 受限使用 | **最多1次，≤0.3s** | 只允许在打斗中间使用（锁定目标/杀意闪过/被击中瞬间的反应），禁止用于起势和收尾 |

**铁律**：
- 特写必须包含人体局部（手腕/肩线/手指/眼神/脚踝），禁止纯物品特写。
- 环境特写必须由战斗动作触发（箭射出→钉木头→特写箭尾颤抖），不能凭空出现。
- 全片战斗特写占比 ≥ 60%，表情特写 ≤ 10%。

### 3.3 反模式（严禁出现在脚本中）

```
❌ "0-2s 全景缓推→snap zoom眼部特写，瞳孔微缩"  ← 慢镜头摆拍
❌ "风吹竹叶沙沙声，她缓缓抬眼"  ← 文艺收尾，不是打斗
❌ "0.5秒慢放头发飘散"  ← 这不是战斗特写
❌ "镜头环拍360°展示她的身姿"  ← 摆pose，浪费时间
```

## 四、脚本输出格式（强制执行）

每一段打斗脚本必须按以下格式编写，缺一不可：

```
## 第N段 [Xs-Ys]｜段名

**动作**：
- [主方] [物理动作描述：身体部位+方向+力度+接触点]
- [敌方] [物理动作描述：身体部位+方向+力度+接触点]
- 环境反馈：[由战斗动作触发的物理反应]

**运镜**：[机位+距离+方向+速度，一种运镜，不超过10个字]
**特写**：[类型(战斗/环境/表情)] + [具体画面，≤0.5s]
```

### 格式约束：
- 每段只写1种运镜，不要堆砌（"侧跟→甩镜→仰拍" 改为选其中最有效的一个）。
- 动作描述用物理量，不用比喻（"右脚蹬竹竿向左45°弹起3米" ✅ vs "如白鹤掠水" ❌）。
- 敌人必须写具体动作，不能只写"刀手压上"——要写几个、站哪、怎么出招。
- 环境反馈必须由动作触发，不能凭空出现。

## 五、轻功规则（用户验证版，默认启用）

### 5.1 物理反馈强制化
- 严禁凭空飞行或无视重力的漂移。
- 轻功本体必须在 0.3-0.6秒内与环境产生强烈物理交互。
- 关键词：脚踏瓦片腾空、足尖点破水面激起涟漪、落点炸开水花、石阶踩出滑痕、衣摆剧烈回弹、切碎薄雾、风压导致火焰偏转。

### 5.2 动作链与动能连续性
- 动作链固定为："脚点地瞬起 → 空中变线 → 落地接招"。
- 全程保持惯性移动，绝对禁止站桩或原地停顿。
- 强制空间跨度：每2秒至少一次3米以上位移。

### 5.3 残影与本体解耦
- 允许多重运动模糊残影，但仅一个本体有效。
- 只有本体产生真实受力、碰撞与破坏。
- 残影绝对不能独立发动攻击。

### 5.4 运镜辅助
- 优先长镜头跟随（Tracking Shot）或中景侧跟。
- 配合背景极速后退的动态模糊（Motion Blur）增强速度感。
- 禁止用复杂运镜（环绕360°/急甩/鬼畜跳切）拍轻功，这会暴露肢体瑕疵。

## 六、敌人调度规则

### 6.1 敌人不是背景板
- 写了多少敌人，就必须写多少敌人的**站位**和**具体动作**。
- 禁止"黑影从竹林深处浮现"这种模糊描述 → 改为"3名刀手从竹林左前方7米处呈扇形逼来，2名弩手从右侧竹架上居高射箭"。

### 6.2 群战调度
- 敌人行动必须有逻辑：谁先出手、从哪个方向、用什么招、主角怎么应对。
- 禁止回合制：主角一次打一个、其余人站着等 = 兵乓球赛。
- 正确做法：多人同时压迫 → 主角用走位/环境/轻功破阵 → 敌人互相挡路/误伤/阵型崩溃。

### 6.3 弓手/弩手特别规则
- 远程武器不能在近距离（<5米）写"完整拉弓实射"。
- 弩箭必须有弹道（从哪射出、经过什么、钉在哪）。
- 主角闪避箭矢时必须交代：箭从哪来→怎么躲→箭钉在哪→钉点特写。

## 七、镜头语言规则

### 7.1 运镜选择（按场景选一种，不要堆砌）

| 场景 | 推荐运镜 | 禁止 |
|------|---------|------|
| 对穿抢位 | 中景侧跟 | 正面固定机位 |
| 轻功腾空 | 低机位仰拍/贴地跟 | 俯拍（看不清速度） |
| 近身破招 | 贴身短焦（武器距离内） | 大全景（看不清细节） |
| 群战调度 | 高机位俯拍/中景跟主角 | 频繁切镜（会混乱） |
| 史诗高潮 | 大远景俯拍（7-10s区间，1-2s） | 起势/收尾用大远景 |
| 收尾定格 | 缓拉中景 | 急甩/跳切 |

### 7.2 切镜频率
- 15秒片：3-5次切镜为佳，不要超过7次。
- 每次切镜必须有信息增量（新角度/新动作/新环境反馈），不能"为了切而切"。
- 同一动作段落内（3秒）尽量保持同一运镜，不要在0.5秒内换3种镜头。

## 八、地形环境规则

不同地形必须与打斗结合，不能只是"背景板"：

| 地形 | 打斗结合方式 | 环境反馈示例 |
|------|------------|------------|
| 竹林 | 踩竹借力/竹竿做掩体/削竹为刃 | 竹折雪落/箭钉竹竿回弹/刀痕劈竹飞溅 |
| 雨巷 | 墙壁蹬转/屋檐借力/水洼滑步 | 水花炸溅/脚踩积水水幕/雨被剑风切开 |
| 沙漠 | 沙丘变高差/沙尘遮视线/马背借力 | 沙尘暴/蹄扬黄沙/刀劈沙面沙浪 |
| 雪原 | 踩雪留痕/冰面滑行/雪崩埋人 | 雪粉炸溅/冰裂/雪崩连锁 |
| 客栈 | 桌椅翻飞/房梁借力/门窗破入 | 木屑飞溅/碗碟碎裂/烛火熄灭 |
| 密林 | 树干绕行/藤蔓借力/树枝遮蔽 | 树皮削飞/落叶旋涡/枝断坠落 |

## 九、情绪×动作映射

| 情绪状态 | 动作质感 | 节奏 | 运镜倾向 |
|---------|---------|------|---------|
| 试探 | 轻触即退、测距为主 | 中速、有呼吸感 | 中景、平稳 |
| 杀意 | 最短路径、一击必杀 | 极速、无犹豫 | 贴身短焦、急停 |
| 绝望 | 拼命式、不计代价 | 狂暴但不失控 | 手持晃动、低机位 |
| 冷静 | 最优路径、借力打力 | 从容但致命 | 中景侧跟、平稳 |
| 愤怒 | 力量型、大开大合 | 猛烈但有章法 | 低机位、跟拍 |

## 十、角色对抗分工（已锁定）

- **顾栖月**：近身=轻功+腿法+玉笛；远程=吹笛压节奏/扰步点（清弦引）。
- **萧烬弦**：近身=寂灭指+擒拿手；远程=弓箭封线/点杀。
- **近距规则**：近距优先擒拿/指法/控线，不写"贴脸完整拉弓实射"。

## 十一、可复用母版（直接套）

### E1 双人极速对抗母版（无第三人）
场景：{同一空间，不换场}。人物：{A}与{B}双人对抗，禁止第三人下场。
视频要求：15秒，24fps，16:9，35mm胶片颗粒，电影级低调摄影，真实体积光与环境反射，无字幕无字卡无水印无Logo，无现代元素，无符文神光，无无动机跳切。
镜头：主机位中景侧跟，少切镜（3-5次），前2秒强制高速起手，慢放最多1次0.2-0.35秒且必须打在打斗中间。
动作：
- 0-2s 高速对穿抢中线，首次接触即交锋
- 2-5s 远距封线与步点扰动，连续攻防交换
- 5-8s 近身三连破招，慢放0.3s打在第2招接触瞬间
- 8-11s 核心破局（内劲对冲或擒拿反制）
- 11-13s 半招压制
- 13-15s 收势定格
音频：无BGM无对白，仅场景音与兵器拟音。

### E2 一对多母版（群战）
场景：{同一空间含掩体/地形}。人物：{A} vs {N名敌人}。
视频要求同E1。
规则：
- 敌人必须标注数量、站位、武器、出场顺序
- 禁止回合制（一次打一个、其余等着）
- 主角必须用走位/环境破阵，而非逐个击破
- 弓手/弩手必须有弹道描写
镜头：中景跟主角为主，1次高机位俯拍展示敌阵，特写均打在战斗瞬间。
音频：同E1。

### E3 空中对穿母版（轻功强化）
场景：{同一空间含可借力点：柱/栏/梁}。人物：{A}/{B}。
视频要求同E1。镜头：低机位仰拍为主，1次中景侧跟。
动作：
- 0-2s 同步蹬地起跳并首次交错
- 2-5s 空中远程互压与封线
- 5-8s 二次腾空变线破角
- 8-11s 落地即贴身反制
- 11-13s 双人对峙
- 13-15s 收尾
要求：每次起落都要有落点反馈，禁止漂浮。

## 十二、反错清单（每次出稿前逐条自检）

### 格式自检
- [ ] 每段是否按"动作/运镜/特写"三栏格式编写？
- [ ] 动作描述是否全部为物理量（无文学修辞）？
- [ ] 每段运镜是否只写了1种（没有堆砌）？

### 时间自检
- [ ] 前2秒是否已有打斗接触（非摆pose）？
- [ ] 2-13秒是否每2秒至少1次攻防交换？
- [ ] 慢镜头是否只有0-1次，且打在打斗中间？
- [ ] 史诗镜头是否只有0-1次，在7-10s区间，且包含人物？

### 特写自检
- [ ] 战斗特写占比是否 ≥ 60%？
- [ ] 表情特写是否 ≤ 1次且 ≤ 0.3s？
- [ ] 每个特写是否包含人体局部（非纯物品）？

### 动作自检
- [ ] 是否有"一人站着不动另一人出招"的回合制？
- [ ] 是否有"跑动蓄势"（应改为滑步/垫步/切步/蹬转）？
- [ ] 弓手/弩手是否在近距离写了"完整拉弓实射"？
- [ ] 武器是否出现串位/断裂/变形/互换？
- [ ] 敌人是否有具体站位和动作（非"黑影浮现"类模糊描述）？

### 环境自检
- [ ] 每段打斗是否有环境微反馈？
- [ ] 环境反馈是否由战斗动作触发（非凭空出现）？
- [ ] 轻功是否有落点物理反馈？
