# 沸腾之雪 · 打斗脚本生成 SOP（流程）

> **规则全部在 VIDEO_LESSONS.md 中，本文件只管流程。**
> 写完脚本后对照 VIDEO_LESSONS.md 逐条自检，通过才交付。

---

## STEP 0：前置准备

1. **读 VIDEO_LESSONS.md**（`scripts/storyboards/VIDEO_LESSONS.md`）— 必读，核心价值文件
2. 确认信息：
   - 场景：地形 + 光照 + 气氛
   - 人物：角色 + 武器（查 `bibles/WEAPONS.md`）
   - 敌方：数量 + 武器 + 站位
   - 时长：15s（默认）
   - 情绪：杀意 / 冷静 / 绝望 / 试探 / 愤怒

## STEP 1：按时间分配写骨架

- 0-2s 起势：环境 + 就位 + 首次接触
- 2-13s 激战：连续攻防，分配视觉资源（慢镜头/史诗镜头/特写配额见 VIDEO_LESSONS.md §2.4）
- 13-15s 收尾：1秒内结束

## STEP 2：逐段写动作

每段（2-3秒）：
1. 先写运镜（一种，见 VIDEO_LESSONS.md §2.1）
2. 写1-2个核心动作
3. 每个动作带位移（从哪到哪，见 VIDEO_LESSONS.md §4.2）
4. 融入环境反馈
5. 涉及轻功必须写离地（见 VIDEO_LESSONS.md §3）

## STEP 3：检查敌人调度

- 所有敌人有站位和动作？（见 VIDEO_LESSONS.md §6.1）
- 不是回合制？（见 VIDEO_LESSONS.md §6.2）
- 弓手有弹道？（见 VIDEO_LESSONS.md §5.3）

## STEP 4：转交付格式

- 纯文本不分段不换行
- 无卡点标注、无对白
- 格式见 VIDEO_LESSONS.md §八

## STEP 5：自检（对照 VIDEO_LESSONS.md 全文）

- [ ] 时间分配合规（§一）
- [ ] 镜头/特写/慢镜头配额合规（§二）
- [ ] 轻功写离地（§三）
- [ ] 动作密度≤2个/段（§4.1）
- [ ] 每个动作有位移（§4.2）
- [ ] 武器物理合规（§五）
- [ ] 敌人调度合规（§六）
- [ ] 收尾≤1秒（§七）
- [ ] 交付格式合规（§八）
