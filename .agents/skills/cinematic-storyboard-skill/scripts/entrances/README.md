# 《沸腾之雪》天榜人物出场素材库

这个目录不直接绑定 `scripts/canon/` 的正剧节奏。

用途只有一个：先把重要人物的首次亮相剧情单独沉淀出来，方便后续拿去做：

- 单人角色素材
- 人物预告片
- 天榜人物系列短视频
- 正剧前置引流片段
- 后续正片插入式补亮相

## 当前口径

这里优先服务“还没在正剧里正式打出记忆点”的人物。

- `天榜第一_慕北歌.md`
  慕北歌的“真正天榜第一亮相”。前面有人物关系戏，但还缺一次压住全场的正式亮相。
- `天榜第五_慧明.md`
  以 `bibles/WORLD.md` 为准，慧明是当前总榜口径里的关键未正式亮相人物。
- `天榜第六_未知者.md`
  天榜第六，暂不落真名，先保留代号型出场。
- `天榜第八_陆听潮.md`
  陆听潮可先做复仇向单人素材，不必等主线推到影线总爆发。
- `天榜第十_燕照绫.md`
  燕照绫适合先做西燕处刑人风格短片，先立狠劲和西燕气味。

## 使用原则

- 先让人物改变局势，再揭身份。
- 先做出场记忆点，再补世界信息。
- 一支素材只抓一个核心情绪，不要把人物生平一次讲满。
- 所有出场都遵守现有世界观硬规则：写实高武、无神光、无瞬移、无超自然化。

## 后续可扩展

如果后面要继续补，可以沿这个目录继续加：

- 已在正剧里出现，但还缺“个人封神亮相”的角色
- 反派阵营副手
- 黑市执事、万宝楼评榜人
- 非天榜但有传播价值的关键人物
