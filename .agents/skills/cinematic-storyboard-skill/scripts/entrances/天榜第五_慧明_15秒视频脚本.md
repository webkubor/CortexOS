# 慧明｜枯树护童｜15秒视频脚本

场景：北荒雪山山道旁，一棵被风雪吹枯的老树立在半坡，地面积雪深厚，寒风持续横卷，树下有一方被坐得微微塌陷的雪地，一名小男孩缩着肩站在不远处，远处山道上一队快马裹雪疾驰而来，整片空间保持单一连续地形
人物：慧明、小男孩、骑快马的江湖人、远处围观路人
视频要求：15秒，24fps，16:9，35mm胶片颗粒，电影级低调摄影，真实体积光与环境反射，无字幕无字卡无水印无Logo，无现代元素，无符文神光，无无动机跳切。
镜头要求：非一镜到底，允许5次切镜；前段先建立雪山、枯树与静坐三日的寒意，中段给小男孩近距离发问和慧明的极简回应，后段集中表现快马冲踏与雪莲般护体的物理奇观，结尾停在众人惊散后的静态画面
剧情：0-3秒：大雪压山，镜头从枯树干缓慢移到树下，只见慧明独自坐在雪地里，僧袍落雪，肩头结霜，小男孩站在一旁缩着脖子，小声问：“你已经在这三天了，这大冬天的不冷吗？”；3-6秒：慧明缓缓睁眼，只说“修行。”男孩又往前半步，仰头问：“大和尚，我爹说武功好的人不怕冷，你是武林中人吗？”慧明平静回答：“不是。”；6-9秒：远处山道忽然传来急促马蹄声，一队快马卷着雪雾直冲枯树下，骑手毫无减速，马蹄眼看就要踏到慧明和孩子；9-12秒：慧明不躲不闪，低声一句“阿弥陀佛。”他周身的风、雪、寒雾与地面积雪同时受力鼓起，沿他与男孩四周翻卷成层层白色弧面，看上去如一朵骤然盛开的雪莲；12-15秒：前排快马撞上这圈雪莲般的受力雪幕，连人带马整股被掀翻弹飞出去，后排骑手惊慌勒马，周围路人四散奔逃，摔落的骑手和外围人群失声大喊“妖僧！”，而小男孩只是愣在原地抬头看着慧明，雪幕慢慢散回风里
动作规则：每次攻击或动作都要完整呈现 起手蓄势→发力路径→命中反馈→收势；角色通过惯性连续移动，无站桩；内力必须有触发动作与传播轨迹，不可凭空生效；本片核心是极致防御与稳态护体，雪莲视觉必须由风、雪、寒雾、地面受力和空气折痕共同形成，不得写成发光法相。
连续性约束：同一角色在同一时段仅允许一套肢体拓扑与一件主武器；慧明全程不持刀、不握剑、不突然改用兵器；马队数量、马具材质与冲刺方向保持一致，不得凭空增减。
轻功默认锁定：严禁凭空飞行或无视重力漂移；慧明全程不做飞身闪避，只以原地稳坐、起身、定步完成防御；护体效果通过雪层鼓起、寒风回卷、僧袍回摆和脚下积雪压实来体现。
对抗反馈：本片的压制感通过“快马全速冲来却撞不开一个站着不动的和尚”来体现；人和马被整体反推、失衡、翻倒、滑出雪地，避免任何玄幻冲击波与无来源爆炸。
环境反馈：枯树枝被风压得吱响，小男孩鞋边积雪被马蹄震起，雪幕形成时地面积雪层层翻卷，撞上的前排快马马蹄离地、胸口受阻、整匹马侧翻滑出，落地后雪粉炸散，最后雪花重新回落到慧明肩头。
音频：全程无BGM，仅保留山风、落雪、枯枝摩擦声、远近马蹄声、马鼻喷气声、衣袍受风声、雪幕鼓起时的低沉压风声、马匹翻倒撞雪声、小男孩前段童声提问与后段短促吸气声、摔落骑手和外围路人的惊叫“妖僧”；慧明台词语速平缓，像在说一件与风雪一样平常的事。
结尾：镜头停驻在枯树下，慧明与小男孩仍在原地，翻倒的人马散在雪地远处，不收黑，只留风雪与惊叫余音被山风吹远。
