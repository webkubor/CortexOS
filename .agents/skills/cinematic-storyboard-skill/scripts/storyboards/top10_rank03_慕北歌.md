# 《沸腾之雪》天榜十绝·天榜第一·慕北歌（15s）正式版

## 🎬 核心设计理念
- **视觉逻辑**：**“孤傲与透明”**。画面追求极致的干净。强调剑尖划过的空气真空痕迹与极寒带来的物理折射。色彩以纯白、冰蓝、竹青为主。
- **动态节奏**：极简。去繁就简，动作如冰晶折射般不可捉摸，强调“切削”而非劈砍。
- **环境背景**：深山竹林，被暴雪覆盖。竹枝挂冰，空气仿佛被冻结。

## 🎥 镜头分解 (15s)
写实高武武侠短片，15 秒，24fps，16:9，35mm 胶片颗粒，电影级光影，无字幕水印 Logo 及现代元素，无法阵符文神光，无跳切瞬移。场景为深山雪竹林，竹枝挂冰，风雪吹过，同一地形连续作战。主角参考 @sword-god-mu，身披纯白素绸长袍，外罩雪狼毫大氅，腰间悬挂神兵“问雪”，剑鞘如白犀角温润，剑身全透明如坚冰，具备极低温物理属性，靠近剑刃的雪花会被无形锋芒切开。刺客为青灰劲装，持长剑，身形敏捷。音效含极度寂静中的冰裂声、微弱寒风啸叫、清冽金属出鞘声、极细高频切开声（Silk-cutting）、万千冰晶落地清脆声、还剑入鞘“咔”响。
0-3.0 秒，寂：白衣问雪。镜头为远景拉近至侧脸特写，宽画幅全景展现雪竹林肃杀感。主角立于弯曲冰竹之上，竹身因重力微颤，风雪漫天，主角发丝微动眼神坚毅，右手轻搭“问雪”剑柄。
3.0-6.0 秒，起：剑气空旋。镜头为微距特写捕捉拇指拨开剑格，透明剑身出鞘瞬间。剑身周围空气由于极低温产生肉眼可见的折射波纹，落下的雪花在触碰剑刃前被整齐切开，冰凌崩碎。
6.0-9.0 秒，切：真空碎影。镜头为横版中景接升格，捕捉极简切削动作。面对围攻，主角仅仅横向一抹，剑尖划过处留下一道细长透明真空空气槽，三名刺客咽喉瞬间出现红线并迅速封冻，无鲜血喷溅。
9.0-12.0 秒，落：冰裂回响。镜头为地面低角度特写接极度拉远。主角脚尖轻点雪面落地，周围十米内竹叶积雪与冰凌受剑气余波感应，在同一时间全部崩碎化为晶莹碎屑，如碎铃落地。
12.0-15.0 秒，定：剑归寂灭。镜头为全景定格背影，缓慢后拉。主角背对镜头，侧首还剑入鞘，身形与漫天风雪融为一体，动作干净利落。
15.0 秒，全景收尾，风雪声瞬间回归，盖过一切。
