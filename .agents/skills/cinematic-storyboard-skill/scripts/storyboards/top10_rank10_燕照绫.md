# 《沸腾之雪》天榜十绝·天榜第十·燕照绫（15s）正式版

## 🎬 核心设计理念
- **视觉逻辑**：**“大漠孤烟与赤练火花”**。画面追求粗粝的质感。强调钢节长鞭与地面/兵刃摩擦产生的物理火花及沙尘爆炸效果。色彩以大漠黄、赤红、玄铁色为主。
- **动态节奏**：灵动且狠辣。长鞭轨迹如灵蛇，动作起手-轨迹-命中-反馈完整，强调惯性与内力爆发的真实感。
- **环境背景**：关外大漠古驿道，残破烽火台，风蚀岩。漫天黄沙，风声凄厉。

## 🎥 镜头分解 (15s)
写实高武武侠短片，15 秒，24fps，16:9，35mm 胶片颗粒，电影级光影，无字幕水印 Logo 及现代元素，无法阵符文神光，无跳切瞬移。场景为关外大漠驿道，残破木栏，断裂旗杆，漫天风沙，沙粒击打木材声清晰，同一地形连续作战。主角参考 @wild-beauty-yan，野性美感，发型利落，身着紧身皮革与丝绸拼接劲装，勾勒身段，佩戴狼牙坠饰，手持钢节长鞭“赤练”，鞭体具备真实金属重量与反弹逻辑。刺客为西燕刀手，劲装厚甲，杀气凛冽。音效含风沙呼啸、鞭节金属颤鸣、长鞭破风脆响、兵刃碰撞火星声、木架崩裂、重物坠沙闷响。
0-3.0 秒，起：赤练拖沙。镜头为低机位贴沙极速推进，宽画幅展现大漠空旷。主角侧身而立，长鞭鞭节拖曳于沙地，划出弧线火花。主角半转身抬腕，内力灌注鞭梢，破空抽裂前方风沙，气流激荡。
3.0-6.0 秒，缠：三路围杀。镜头为中景横拍接环绕。面对三路刀手合围，主角不闪不退，长鞭如红影横扫。捕捉长鞭锁住敌腕、借力反拉将敌人掼撞在残破木桩上的物理过程，布料撕裂同步反馈。
6.0-9.0 秒，崩：沙尘破影。镜头为升格特写接动态追踪。主角足尖点踏断车辕腾身，空中拧腰挥鞭。鞭尾击碎袭来长刀，火星四溅，落地瞬间鞭梢击地引发小规模沙粒爆炸，逼退偷袭者。
9.0-12.0 秒，锁：困兽之斗。镜头为穿透式近景跟拍。长鞭使出缠、锁、拉、崩四式连环，动作干净利落。捕捉金属鞭节勒入敌人护甲产生变形、木屑崩飞的细节，强调内力爆发。
12.0-15.0 秒，定：漠影远去。镜头为大远景缓慢拉近至主角侧脸定格。主角帅气收鞭，不恋战，翻身上马沿驿道疾驰而去，背后是残旗猎猎。画面定格在**【炼】**字视觉符号上。
15.0 秒，全景收尾，风沙渐覆战场，只余马蹄声渐远，BGM 重鼓落定。
