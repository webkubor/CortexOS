# 《沸腾之雪》天榜十绝·天榜第七·萧烬弦（15s）正式版

## 🎬 核心设计理念
- **视觉逻辑**：极致的“冷色调”与“金属质感”。强调箭簇的寒光、指尖的稳健、以及鲜血溅在雪地上的高饱和反差。
- **动态节奏**：0-9秒为极静的蓄力与远射，9-12秒为极速的近身瞬杀，12-15秒回归死寂。
- **环境背景**：苏王府飞檐之上，大雪封路，瓦片覆冰。

## 🎥 镜头分解 (15s)
写实高武武侠短片，15 秒，24fps，16:9，35mm 胶片颗粒，电影级光影，无字幕水印 Logo 及现代元素，无法阵符文神光，无跳切瞬移。场景为苏王府后院雪池，红墙黑瓦积雪盈寸，漫天大雪，长廊灯笼在寒风中摇曳，雪粉从瓦片边缘震落，同一地形连续作战。主角参考 @wang-fu-archer，手持神兵乌麟玄角长弓，弓两端收尖成利刃可作短矛，背负重型破甲重箭，指法通神，具十象之力，两指可扣断敌刃。刺客为暗色劲装，持长枪刀剑，身形利落杀机凛冽。音效含风雪呼啸、极速坠落风声、瓦片崩飞、积雪炸开、金属切割火星、指法骨裂、弓弦满拉、箭矢破音障、箭入石爆响。
0-3.0 秒，轻功瞬落飞掠，横版极速俯冲跟拍，模拟自由落体，高速移动营造眩晕感，落地拉宽展现王府雪池全景。主角从王府高耸飞檐垂直坠落，黑色披风烈烈作响形如寒鸦，落地瞬间瞬步侧移，借下坠惯性贴地飞掠十余米，气流冲击使瓦片崩飞、积雪炸开呈环形冲击波。
3.0-7.0 秒，近战弓刃指杀，横版环绕镜头接升格特写，清晰捕捉近身厮杀、弓刃削枪、指法扣喉细节。主角冲入刺客阵，挥舞乌麟玄弓，以弓端利刃削断敌枪，火星四溅，右手寂灭指扣住刺客咽喉迸发暗劲，将其震退数米。
7.0-12.0 秒，远射十象之力，横版升降镜头接慢动作特写再极速快放，捕捉后空翻、滞空满弓、箭矢贯穿全貌。主角借力刺客肩膀后空翻，滞空最高点短暂定格，身体后仰以十象之力将弓拉至满月，一箭破音障，重箭拖曳白色气流，贯穿回廊所有门窗，将百步外重甲统领连人带盾钉死在石狮子上，路径形成真空通道，窗纸木屑炸裂，统领与盾被冲击力带飞五米，石狮子崩裂四散。
12.0-15.0 秒，余韵收势定格，横版低机位仰拍，大远景缓慢拉近至特写。主角单膝落地，两指轻抹弓弦震落血珠碎屑，缓缓抬头，月光掠过冷厉侧脸，眼神如冷铁，背景为炸裂门窗、倒地刺客与破碎石狮，定格于主角眼神。
