# 影视分镜技能 (Production-Grade)

> **核心目标**: 生成专业的 12-15s 分镜脚本与 AI 视频提示词。
> **设计哲学**: 以刀为眼、以速为魂、以暴雨/雪为境。

---

## 1. 核心工作流 (Core Workflow)

### 1.1 角色锁定
在进行任何分镜前，必须调用 `cast/*.json` 的物理描述。
- **苏梦城**: 白发/银面具（江湖线），王袍/重磅锦缎（朝堂线）。
- **顾栖月**: 玉笛/玄色劲装/纯欲清冷。
- **慕夕歌**: 高马尾/直剑/白色剑鞘。

### 1.2 动作设计铁律 (Master Class Actions) [NEW]
在生成分镜脚本时，必须遵循以下“极速暴力美学”：
- **【以刃为眼】 (Weapon-Eye Focus)**：镜头必须贴合武器轨迹进行同步运动（如跟刀拍摄、环绕枪尖）。
- **【动量守恒】 (Total Inertia)**：12s-15s 全程无静止、无停顿。动作之间必须有**惯性衔接**（如拖刀滑行、借力翻身）。
- **【打击卡点】 (Impact Pacing)**：关键碰撞帧采用 **“0.5s慢放 + 极速回正”** 的弹性节奏。

---

## 2. 脚本输出格式 (Storyboard Format)

### A. 场景上下文 (Scene Context)
> **风格**: 8K, Dolby Vision, 写实高武 (低调摄影)
> **音频**: [BGM选用 + 音效设计]
> **故事**: [15秒剧情简述]

### B. 分镜脚本 (Storyboard Text) [NEW - 纯文本/无表格]
**格式规范：[时间段]|[标题·动作核心]**
- **[镜头语言]**: 包含机位、运镜术语 (e.g., *Ultra-wide angle, Snap Zoom, Dolly Push*)。
- **[动作表演]**: 核心动能轨迹（起势、发力、收势）。
- **[视觉反馈]**: 物理奇观（雨幕撕裂、粒子环绕、金属微光、打击慢放反馈）。
- **[音频设计]**: 具体的音效卡点描述。

### C. 综合提示词 (AI Master Prompt) [NEW]
**一个高度描述性的长段落，包含以下模块：**
1. **[画质与影调]**: 8K, HDR10+, 120fps, 35mm胶片颗粒, 暗调冷色。
2. **[角色一致性]**: 角色锁定特征 + 武器描述。
3. **[动作与运镜]**: 全程惯性移动、以刀为眼、急停急启。
4. **[物理奇观]**: 雨丝物理撕裂、真空环绕、寒芒四溅、打击反馈。
5. **[负面提示]**: 无字幕、无水印、无现代元素、无神光符文。

---

## 3. 分镜生成路由 (Routing)
- **极速打戏**: 120fps, 强运动模糊, 超广角。
- **意境开场**: 浅景深, 慢镜头, 自然光, 情绪吟唱。
- **悬疑暗流**: 变焦缩放, 低角度, 环境噪音。

## 4. 平台文案格式规则 (Publishing Format)
- **微信视频号**:
  - 第一行强制写标题行：`《沸腾之雪》第X集｜副标题`
  - 从第二行开始写正文文案。
  - 话题必须使用 `#` 前缀。
- **哔哩哔哩**:
  - 标题格式：`【沸腾之雪·第X集】副标题｜一句钩子`
  - 简介采用“冲突落点 -> 信息落点 -> 下一集钩子”三句式。
  - 标签用逗号分隔，6-10个关键词。
