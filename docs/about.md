## 🧠 关于 CortexOS

**CortexOS** 不仅仅是一个文档库，它是我的**外部大脑 (Exocortex)** 实战原型。

它通过 **MCP (Model Context Protocol)** 标准协议，将零散的知识碎片、严苛的工程规范（Rules）、以及每一次迭代的复盘记忆（Memory）物理锚定在本地。无论我唤醒的是 Gemini、Claude 还是 Codex，它们只需“插上”这块硬盘，就能瞬间继承我的审美习惯与思维逻辑。
