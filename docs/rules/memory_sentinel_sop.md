# 🕯️ envAI 哨兵: 自动记忆更新协议 (SOP)

## 1. 核心定位
**envai-sentinel** 是常驻后台的记忆同步服务，由 PM2 管理，负责实时监听并向量化外部大脑中的 Markdown 文档。

## 2. 监听路径
- **根路径**: `~/Documents/AI_Common`
- **监听文件**: 所有的 `.md` 格式文档。

## 3. 同步逻辑 (吸收 memsearch 理念)
- **触发机制**: 文件保存后 2 秒自动触发（防抖处理）。
- **切片策略**: 优先识别 `##` 标题进行语义分块 (Heading-based Chunking)，确保记忆块语义完整。
- **指纹去重**: 使用内容 SHA-256 生成稳定 `chunk_id`。若内容未变，ID 保持一致，Milvus 不产生冗余数据。
- **混合检索**: 75% 向量检索 + 25% 关键词命中加权，精准捕获武侠专有名词。

## 4. 状态反馈 (macOS Native)
- **静默成功**: 若内容未变，提示“记忆已是最新”。
- **新增通知**: 若有新内容，发出系统推送：“新增了 N 个记忆片段”。
- **日志查看**: 执行 `pm2 logs envai-sentinel` 可查看详细同步轨迹。

## 5. 运维手册
- **启动/重启**: `cd backend && pm2 restart envai-sentinel`
- **查看状态**: `pm2 list`
- **手动补全**: 访问 `http://localhost:3000/manage` 执行全量扫描。
