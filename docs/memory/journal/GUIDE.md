# 执行日志记录规范

> **标准文档**: 见 `memory/journal/README.md`

## 🚀 快速使用指南

### 1. 创建新任务日志

```bash
# 使用模板文件
cp docs/memory/journal/new-task-template.md docs/memory/journal/$(date +%Y-%m-%d).md
```

### 2. 标准日志格式

每个日志文件应包含：

```markdown
# YYYY-MM-DD: 任务标题

## 📋 任务概览
- **任务**: [简要描述]
- **触发 Agent**: [Agent 名称]
- **优先级**: [高/中/低]

## ✅ 已完成
- [x] 步骤1
- [x] 步骤2

## 🔄 进行中
- [ ] 步骤3 - [当前进度详情]

## ⏸️ 待办
- [ ] 步骤4
- [ ] 步骤5

## 📊 资源消耗
- Token: [估算]
- 时间: [开始-结束]

## 🔍 大脑自检记录
- [检查项目]
  - [结果1]
  - [结果2]

## 💡 关键决策
- [记录重要决策或问题]
```

### 3. 切换 Agent 时必做

- [ ] 读取最新日志
- [ ] 更新"进行中"进度
- [ ] 添加"任务概览"中的 Agent 信息
- [ ] 记录"大脑自检记录"检查项

### 4. 完成任务后

- [ ] 标记所有步骤为 [x]
- [ ] 更新"资源消耗"统计
- [ ] 添加"复盘/经验"到 retrospectives/
- [ ] 确认日志可被下个 Agent 读取
