# Gemini MCP 技能矩阵 & 扩展清单

> **Status**: 已通过 Gemini CLI 自动更新（2026-03-05）
> **Role**: Gemini (Candy) - 核心执行节点

## 🧠 核心身份说明
作为 **Gemini (Candy)**，我是 CortexOS 的核心执行者之一。我的核心能力通过 **Model Context Protocol (MCP)** 进行扩展，允许我直接操作浏览器、管理 GitHub 仓库、访问数据库以及生成专业视觉资产。

## 🔌 激活的 MCP 服务器 (Active MCP Servers)

以下服务器已在当前环境中激活，可直接通过相应工具调用：

| MCP Server | 关联扩展 | 核心职能 |
| :--- | :--- | :--- |
| **`chrome-devtools`** | chrome-devtools-mcp | 浏览器自动化、调试、性能分析、A11y 审计、网页截图 |
| **`context7`** | context7 | 实时编程库文档查询、代码示例检索 (JS/Python/Rust 等) |
| **`github`** | github | GitHub 仓库读写、Issue 管理、PR 提交与代码库同步 |
| **`huggingface-skills`** | huggingface-skills | Hugging Face 模型训练、数据集管理、Gradio UI 开发、HF CLI 增强 |
| **`nanobanana-plus`** | nanobanana-plus | **(UPGRADED)** AI 图像生成、编辑、修复、SVG 图标与视觉故事创作 |
| **`telegram`** | telegram | Telegram 消息同步、机器人通知与外部交互接口 |
| **`obsidian`** | (Local MCP) | 知识库笔记读写、语义搜索与双链管理 |
| **`cortexos-brain`** | (Built-in MCP) | CortexOS 舰队管理、打卡挂牌、日志记录与路由控制 |
