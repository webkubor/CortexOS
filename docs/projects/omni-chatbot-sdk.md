# 📦 omni-chatbot-sdk

## 项目定义
一个通用的、高颜值的、轻量级 AI 聊天机器人前端 UI SDK。

## 核心价值 (The Vibe)
- **Glassmorphism**: 毛玻璃高级质感。
- **Zero Conflict**: 强制 pl- 命名空间，绝不污染宿主样式。
- **i18n Ready**: 内置 en/zh 自动识别。

## 维护手册
- **发包**: npm run build && npm publish
- **规范**: 遵循 AI_Common/docs/rules/webkubor_vibe_manifesto.md
