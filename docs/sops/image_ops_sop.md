# 图像生成决策矩阵与 SOP

## 1. 场景路由

根据任务意图自动选择工具链，禁止工具混用。

### A. IP 角色与电影分镜

- **工具**: `edit_image`（nanobanana-plus）
- **前置条件**: 必须指定并加载角色参考图（存放于私有素材目录，不入仓）
- **执行规范**: 如用户在私有记忆中定义了图像风格协议，优先遵循；未定义时按当前任务目标落地
- **归档**: 私有素材目录（由本机环境决定）

### B. 纯技术 / 工程化封面

- **工具**: `generate_image`（nanobanana-plus）
- **模型建议**:
  - 极速预览: `gemini-3.1-flash-image-preview`
  - 高质归档: `imagen-4.0-fast-generate-001`
- **规范**: 统一遵循 `zero humans, zero faces` 与技术主题一致性
- **红线**: 不得偏离工程主题或引入无关人物形象
- **归档**: 私有素材目录（由本机环境决定）

## 2. 完工定义（DoD）

1. 图像已生成并符合对应场景规范
2. 图像已上传并获得可用 HTTPS 链接
3. 图像已重命名并存入指定本地归档目录
4. Markdown 源码已正确引用该链接
