# 🧠 CortexOS 大脑演化史

> 这份文档记录 CortexOS 从“笔记仓库”演进为“多 Agent 协作中枢”的关键里程碑。  
> 目标不是讲故事，而是保留可审计的决策轨迹。

---

## 🏗️ 当前智力档位: v5.4.0 (协作与记忆分层期)

---

## [v5.4.0] - 2026-03-04（当前：协作治理收敛）
- **角色分工上线**：Agent Team 增加 `前端/后端` 角色字段，任务登记从“谁在做”升级为“谁以什么职责在做”。
- **僵尸节点识别**：舰队状态加入 stale 检测，降低看板被历史任务污染的概率。
- **技能管理落地**：新增自动扫描型 Skills 管理页，覆盖安装目录、调用方式、验证步骤，不再停留在概念介绍。
- **影响**：协作层从“可见”进入“可治理”，新人可按手册直接接管技能与任务。

## [v5.3.0] - 2026-03-04（记忆模型重构）
- **记忆公式确立**：形成“助手私有记忆 vs 用户长期记忆”的双层模型，并写入路由协议。
- **日志迁移**：Agent 运行日志迁移到 `.memory/logs/`，不再占用用户记忆空间。
- **人格资产归位**：Candy Manifest、SOUL 合同标准与调教策略从项目通用区迁移为可控资产。
- **影响**：隐私边界与上下文边界清晰化，token 消耗与检索噪音同步下降。

## [v5.2.0] - 2026-03-04（文档架构解耦）
- **规则库瘦身**：`rules/` 仅保留约束、禁令、价值观；操作细节拆分到 `sops/` 与 `ops/`。
- **语义分区完成**：新增 `persona/`、`tech/` 等目录，避免“规则、教程、协议”混放。
- **影响**：检索路径可预测，AI 首屏定位速度提升，误读规则概率下降。

## [v5.1.0] - 2026-03-02（看板可信化）
- **客观进度**：引入基于 `TODO.md` 勾选的进度计算，降低主观汇报偏差。
- **看板交互增强**：Busy/Idle 状态、脉冲与进度动效上线。
- **品牌升级**：从 `AI_Common` 全量切换到 **CortexOS** 命名体系。
- **影响**：团队状态从“口头同步”升级为“可视化 + 可核验”。

## [v5.0.0] - 2026-03-02（MCP 化）
- **MCP Server 上线**：以 FastMCP 暴露核心工具，外部大脑进入可编排调用阶段。
- **工具合约化**：通过结构化 Tool 输入输出降低幻觉调用和参数漂移。
- **影响**：CortexOS 从文档系统进化为可执行系统。

## [v4.7.0] - 2026-03-02（多 Agent 编排协议）
- **入队握手前缀**：统一 `[AI-TEAM][nodeId][#No]`，实现机位可识别。
- **口令移交**：支持自然语言触发队长切换与任务移交。
- **影响**：多机并行从“人肉协商”进入“协议化协作”。

## [v4.0.0] - 2026-02-28（防撞车与凭证隔离）
- **Fleet Guard**：引入 `fleet:claim` 作为并行任务打卡闸门。
- **凭证隔离初版**：建立高敏资产外置原则，后续收敛为 `~/Documents/memory/secrets`。
- **通知链路**：接入 Lark + 系统通知双通道。
- **影响**：并行冲突与凭证泄漏风险被首次系统化约束。

## [v3.8.0] - 2026-02-28（技能云端化与检索重构）
- **技能解耦**：SCM、XHS 等能力拆分为独立仓库，主仓聚焦中枢编排。
- **本地 RAG 切换**：从远程 Milvus 迁移到 ChromaDB + Ollama。
- **影响**：主仓复杂度降低，技能可独立演进与发布。

## [v3.0.0] - 2026-02-20（协议成形）
- **SSOT 确立**：`router.md` 成为唯一入口，Agent 冷启动必须先读路由。
- **部门化协作**：Core、Writers、Ops 等职责首次明确。
- **影响**：从“人依赖流程”转向“流程约束人”。

## [v2.0.0] - 2026-02-02（基础设施期）
- **哨兵机制**：建立任务日志与定时同步雏形。
- **VitePress 成型**：知识可视化标准建立。
- **复盘机制**：复盘从项目内散落记录，逐步走向分层归档。
- **影响**：大脑开始具备持续维护能力。

## [v1.5.0] - 2026-01（自动化萌芽）
- **脚本化起步**：`scripts/` 与日志脚本出现。
- **规则雏形**：首批工程规则文件成形。

## [v1.0.0] - 2025-12（仓库锚定）
- **仓库建立**：CortexOS（原 AI_Common）完成物理落盘。
- **目录骨架**：`docs/` 结构确立，摆脱零散笔记模式。

## [v0.5.0] - 2025-11（外部大脑雏形）
- **长期记忆意识**：开始把“经验”当作可复用资产管理。
- **早期规则集**：最初的编码与协作规范写入文档。

## [v0.1.0] - 2025-10（原生笔记期）
- **单机记录**：以 Markdown 记事为主，尚无多 Agent 协作。
